### Rust の `&` と `ref`

`ref` を知らなかったので、以下の issue を見る：

https://github.com/rust-lang/rust-by-example/issues/390#issuecomment-69658337

以下２つは同じ意味とのこと：

```
let x = &y;
let ref x = y;
```

`&` だけで何故ダメなのかわかっていない。

Zephyr のドキュメント読み。
tickless の kconfig が[言及されていた](https://docs.zephyrproject.org/latest/kernel/timeutil.html)ので、自宅の Linux サーバーが tickless か調べる：

```
toyama@blue:~$ grep NO_HZ /boot/config-$(uname -r)
CONFIG_NO_HZ_COMMON=y
# CONFIG_NO_HZ_IDLE is not set
CONFIG_NO_HZ_FULL=y
CONFIG_NO_HZ=y
```

少なくとも、
カーネルのビルド設定としては tickless になっているっぽい。
